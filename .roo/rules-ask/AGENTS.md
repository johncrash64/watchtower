# Project Documentation Rules (Non-Obvious Only)

- Output markdown files numbered00-06 for sort order - each has specific purpose (source, guide, scriptures, terms, applications, threads, timing)
- `study_notes.json` structure mirrors output files - `meta`, `paragraphs`, `terms`, `threads`, `applications`, `timing` keys map directly to generators
- All content is in Spanish - Bible references use Spanish book names (Romanos, not Romans)
- WOL URLs use `/es/` language path and `r4/lp-s` repository codes for Spanish content
- Markdown files cross-reference each other via relative links - renaming breaks internal navigation
